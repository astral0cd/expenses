# Лабораторная работа: Учёт личных расходов

## Цель
Реализовать консольное приложение для учёта личных расходов.

---

## Описание
Программа `expenses.py` позволяет пользователю записывать свои расходы, разбивать их по категориям и анализировать. 

Она работает через аргументы командной строки.

---

## Основные команды
- **Добавить расход:** `python expenses.py add <стоимость> <категория> <название>`
- **Добавить категорию:** `python expenses.py add-category <категория>`
- **Показать все расходы:** `python expenses.py list [категория]`
- **Показать сумму расходов:** `python expenses.py total [категория]`

---

## Требования
- Данные сохраняются между запусками программы.
- Проверка существования категории при добавлении расхода.
- Корректная обработка ошибок (неверный ввод).
- Использование стандартных библиотек Python.

---

## Хранение данных
Формат хранения - **JSON**. Данные разделены на два модуля для удобства: `expenses.py` (интерфейс) и `storage.py` (логика сохранения).

---

### Как запустить
1. Убедитесь, что у вас установлен Python.
2. Склонируйте репозиторий.
3. Зависимости не требуются (используются только стандартные библиотеки).
4. Запускайте команды через терминал, как указано выше.
